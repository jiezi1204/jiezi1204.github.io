DTD文档模型和HTML基础

DTD文档模型分为三种：
1 过渡的（transitional）： 要求非常宽松，继续使用HTML4.01的标识；
2 严格的（strict）： 不能使用任何层的标识和属性，如；
3 框架的（frameset）： 页面中包含有框架，需采用这种DTD；
常用的HTML标签和属性
1 设置文档编码
<meta http-equiv="content-Type" content="text/html;charset=UTF-8"/>
2 设置关键字和描述方便SEO
<meta name="keywords" content="关键字">
<meta name="description" content="关键字">
3 设置移动端开发缩放比例
<meta name="viewport" content="width=device-width,initial-scale=1"/>
content属性值:
width: 可视区域的宽度，值可为数字或关键字device-width；
height: 同width；
initial-scale: 页面首次被显示是可视区域的缩放级别，取值1.0则页面按实际尺寸显示，无任何缩放；
maximum-scale： 最大缩放；
user-scale: 是否可对页面进行缩放，no禁止缩放。
